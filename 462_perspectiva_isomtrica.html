4.6.2. Perspectiva isométrica

1. Introducción

El nombre de isométrico viene de que significa “de la misma medida”, y se refiere a que los tres ejes que usamos para dibujar tienen el mismo ángulo entre ellos: 120°. Las medidas que se toman sobre cada eje se ven reducidas en una proporción de 4:5 (0,82), pero como esta reducción es igual para todos los ejes, podemos dibujar sin aplicarla y el resultado será proporcional.

 

Lectura facilitada

اسم "الإيزومتري" يأتي من معناه "نفس القياس"، ويشير إلى أن الثلاثة محاور التي نستخدمها في الرسم لديها نفس الزاوية بينها: 120 درجة. القياسات التي نأخذها على كل محور تتناقص بنسبة 4:5 (0.82)، ولكن نظرًا لأن هذا التصغير متساوي لجميع المحاور، يمكننا الرسم دون تطبيقه والنتيجة ستكون متناسبة.

2. Descubriendo

Para realizar correctamente la perspectiva isométrica sigue los siguientes pasos:

  1. Elige un punto de vista desde el que quieres ver el objeto. Normalmente, se coloca el objeto sobre un plano horizontal, y se mira desde una esquina superior.
    Traza tres ejes perpendiculares entre sí, que formen ángulos de 120º. Estos ejes se llaman isométricos, y corresponden a las direcciones principales del objeto. El eje vertical se suele llamar Z, y los otros dos ejes se llaman X e Y.
  2. Proyecta las aristas del objeto sobre los ejes isométricos, usando una escala adecuada. Ten en cuenta que las aristas paralelas a los ejes se mantienen iguales, pero las aristas oblicuas se acortan según un factor de reducción. Este factor depende del ángulo que formen las aristas con el plano horizontal, y se puede calcular con una fórmula o con una tabla.
  3. Une los puntos proyectados para obtener las caras del objeto. Distingue entre las caras visibles y las ocultas, usando líneas continuas o discontinuas respectivamente. También puedes aplicar algún tipo de sombreado o color para dar más realismo al dibujo.

Lectura facilitada

Para realizar una perspectiva isométrica correctamente, sigue estos pasos:

1. Elige un punto de vista para observar el objeto, generalmente desde una esquina superior.

2. Dibuja tres ejes perpendiculares entre sí, formando ángulos de 120º. Estos son los ejes isométricos: Z (vertical), X e Y.

3. Proyecta las aristas del objeto sobre los ejes isométricos, ajustando la escala. Las aristas paralelas a los ejes se mantienen igual, pero las oblicuas se acortan según una fórmula o tabla.

4. Conecta los puntos proyectados para formar las caras del objeto. Usa líneas continuas para caras visibles y líneas discontinuas para caras ocultas. Puedes agregar sombreado o color para mayor realismo.
